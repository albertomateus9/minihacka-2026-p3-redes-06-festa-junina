# Festa Junina da Escola

Site estático do projeto **Festa Junina da Escola**, desenvolvido para a MiniHacka 2026-1 da turma do 3º período de Redes de Computadores (Integrado).

## Ideia

Um site para informar os alunos sobre horários, combinados, atrações e mudanças relacionadas à festa junina da escola.
